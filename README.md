# Sistema de Gerenciamento de Usuários (Java Console CRUD)

Este é um projeto de estudo desenvolvido em **Java**, focado na criação de um sistema CRUD (Create, Read, Update, Delete) via terminal. O objetivo principal foi praticar a lógica de programação, manipulação de coleções, tratamento de exceções e arquitetura em camadas.

## Funcionalidades

O sistema permite realizar as seguintes operações através de um menu interativo no console:

- **[1] Cadastrar:** Adiciona um novo usuário com Nome, E-mail e Data de Nascimento.
- **[2] Atualizar:** Modifica os dados de um usuário existente.
- **[3] Excluir:** Remove um usuário da base de dados pelo ID.
- **[4] Buscar por ID:** Exibe os detalhes de um usuário específico.
- **[5] Listar:** Mostra todos os usuários cadastrados.
- **[6] Sair:** Encerra a aplicação.

## Tecnologias Utilizadas

- **Java 11+** (Uso de `var`, `LocalDate`, `Switch Expressions`).
- **Programação Orientada a Objetos** (Classes, Interfaces, Enums).
- **Tratamento de Exceções** (Custom Exceptions).
- **Java Time API** (Manipulação de datas).
